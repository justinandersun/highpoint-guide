# Highpoint Guide
An interactive guidebook for climbing the 50 U.S. highpoints.
See it live @ https://highpoint.guide

## Tech Stack
This project was built using Gatsby.js and GraphQL.

## Requirements
Highpoint Guide requires: 
- `gatsby-cli` version 4.23.0
- `npm` version 8.19.2
